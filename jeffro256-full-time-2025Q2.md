## What

The last quarter I [integrated](https://github.com/seraphis-migration/monero/pull/19) [Carrot](https://github.com/jeffro256/carrot/blob/master/carrot.md) into `wallet2` scanning. I also [integrated](https://github.com/seraphis-migration/monero/pull/32) Carrot over FCMP++ transaction construction into the CLI wallet `transfer` command. I want to continue this work before the upcoming alpha stressnet on May 21st.

Here is a list of things I will attempt to work on this quarter, in rough order of execution:

- Finish CLI wallet support for Carrot/FCMP++
- Start/finish RPC wallet support for Carrot/FCMP++
- Help GUI wallet support Carrot/FCMP++
- Provide support to existing hardware wallet manufacturors on how to securely support Carrot scanning/spending
- Begin soliciting Carrot core implementation audits
- Provide Rust implementation of Carrot for Serai/Cuprate
- Solicit help for multisig implementations of Carrot
- Help out with the FCMP++ integration wherever I can

## Who

I have been contributing to the Monero core repository for [over two years](https://github.com/monero-project/monero/pulls?page=2&q=is%3Apr+author%3Ajeffro256) with a total of [98 merged commits to master](https://github.com/monero-project/monero/commits?author=jeffro256) thus far, with many open PRs. Over the last few months, I wrote up the [Carrot specification](https://github.com/jeffro256/carrot/blob/master/carrot.md), organized [auditing](https://github.com/cypherstack/carrot-audit), for which the community graciously funded, and [began](https://github.com/monero-project/monero/pull/9559) [implementing](https://github.com/monero-project/monero/pull/9697) it. Carrot will be the main supported addressing protocol post-FCMP++ if all goes according to plan. I also worked on the Seraphis migration project in 2023/2024.
